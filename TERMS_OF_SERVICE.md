# Términos de Servicio

Última actualización: 2026-01-03

## 1. Descripción del Servicio
Social Metrics Automation automatiza la publicación y la recolección de métricas en Facebook, Instagram, TikTok y WhatsApp (Cloud API) mediante una API REST y utilidades CLI.

- No se permite spam, scraping indebido ni abuso de APIs de terceros.
- El usuario es responsable de la precisión y legalidad del contenido que publique.

## 3. Datos Tratados
- Datos de redes sociales: publicaciones, métricas de alcance y engagement provistas por las APIs.
- Datos personales: no se almacenan datos personales de usuarios finales; las peticiones se manejan de forma transitoria.
- Credenciales: los tokens/API keys se almacenan como secretos de servidor para autenticar contra las APIs de terceros.
